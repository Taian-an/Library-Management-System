Library Management System (LMS)A full-stack library management application built with Next.js, React, and MongoDB Atlas. This project was developed as part of the Web Application Development Final Examination (Semester 2-2025).🚀 Project OverviewThis system provides a dual-role interface for managing library resources:Admin Role: Full CRUD operations on books (including soft delete), and management of user borrowing requests.User Role: Browsing available books with filter/search functionality and submitting borrowing requests.Key Technical FeaturesNext.js Migration: Successfully migrated from a Vite-based environment to Next.js to support robust Backend API routes (/api/*).Strict Type Safety: 100% any-free implementation. All data models and API responses are governed by TypeScript interfaces (e.g., IBook, IBorrowRequest).Authentication: Secure login/logout system using JSON Web Tokens (JWT).Soft Deletion: Implemented book deletion logic that preserves data integrity while hiding "deleted" items from regular users.🛠️ Setup & InstallationFollow these steps to run the project locally:1.
